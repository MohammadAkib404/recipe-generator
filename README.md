Daily Dish is an AI-driven recipe generator that creates complete, step-by-step recipes using ingredients provided by the user. Each recipe includes an AI-generated image to enhance clarity and presentation. The platform also explores default recipe categories to streamline discovery.

---

## Features
- Generate full recipes from available ingredients  
- AI-generated image for each recipe  
- Clear, structured, step-by-step instructions  
- Simple and fast user flow  
- Practical focus on everyday cooking  

---

## Tech Stack
- **Frontend:** React  
- **Backend:** Node.js  
- **AI:** Large Language Models (LLMs)  
- **APIs:** AI content and image generation APIs  

---

## System Focus
- Prompt-based recipe and image generation  
- Usability and visual clarity  
- Consistent, readable output  

---

## Use Cases
Daily Dish is useful for:
- Deciding what to cook with limited ingredients  
- Visualizing recipes before cooking  
- Reducing food waste
